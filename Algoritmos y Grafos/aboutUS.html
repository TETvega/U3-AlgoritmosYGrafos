Proyecto #2 - Desarrollo de una Aplicación de búsqueda y grafos

Crear una aplicación web interactiva que permita a los usuarios entender y probar los diferentes algoritmos de búsqueda y grafos mediante una interfaz amigable.

Herramientas a utilizar:
  • HTML
  • CSS
  • JavaScript
Descripción del Proyecto

En este proyecto, el equipo desarrollará una aplicación web que incluirá:

  1. Página de Inicio:
    • Bienvenida al usuario y breve introducción al propósito de la aplicación.
    • Menú con opciones para acceder a los diferentes tipos de algoritmos de búsqueda y de grafos.
  2. Páginas de Algoritmos:
    • Cada algoritmo de búsqueda (por ejemplo: búsqueda lineal, búsqueda binaria, etc.) tendrá su propia página.
    • Cada algoritmo de Grafos(por ejemplo: Dijkstra, Prim, Kruskal, etc.) tendrá su propia página.
    • En cada página, se presentará una explicación del algoritmo seleccionado
    • Habrá un área donde se podrán ingresar los datos para probar el algoritmo.
    • Una sección que muestre el resultado después de aplicar el algoritmo seleccionado a los datos proporcionados
  3. Requisitos Técnicos:
    • Se utilizará HTML para la estructura básica de la página
    • El diseño será estilizado con CSS para hacer la aplicación más amigable visualmente.
    • JavaScript será utilizado para implementar los algoritmos de ordenamiento y la funcionalidad interactiva de la aplicación.
