Algoritmo de Busqueda por Saltos || Jump Search

Jump Search es un algoritmo de búsqueda en el cual, en vez de irse de manera secuencial, salta un grupo de elementos de manera repetitiva hasta encontrar el valor buscado o uno mayor. Si se encuentra el valor, termina nuestro algoritmo. Si llegamos a un valor mayor, retrocedemos de uno en uno hasta llegar al valor buscado. Es importante mencionar que el arreglo debe estar ordenado..

Supongamos que tenemos el siguiente arreglo de 15 elementos: [1, 3, 4, 7, 9, 11, 13, 16, 19, 18, 20, 22, 24, 27, 30]:

  • Necesitamos saber la posición del número 27 dentro del arreglo.

  • Cómo determinar el número de elementos a brincar en cada iteración
    Usualmente sería √N donde N es el total de elementos. En este caso √15 = 3.87Redondeando, serían 4.
    Con este tamaño de paso y este arreglo en particular, el peor escenario serían 5 iteraciones para encontrar el número 27.

Estos son solo algunos ejemplos de algoritmos de búsqueda. Cada uno tiene sus propias ventajas y desventajas, y la elección del algoritmo adecuado depende del problema específico que se esté abordando y las características de los datos involucrados.

Código de la función de búsqueda de Saltos para JS

Codigo utilizado para JS

Recibe un numero entero y una lista de numeros enteros ordenada de menor a mayor: ( 13 , [0, 1, 1, 2, 3, 5, 8, 13] )

Se detiene en el momento que encuentra el numero buscado.

                        
                            function busquedaPorSaltos(arreglo, valorBuscado) {
                                const longitud = arreglo.length;
                                let salto = Math.floor(Math.sqrt(longitud));
                                let previo = 0;
                              
                                while (arreglo[Math.min(salto, longitud) - 1] < valorBuscado) {
                                    previo = salto;
                                    salto += Math.floor(Math.sqrt(longitud));
                                        if (previo >= longitud) {
                                            return -1; // El elemento no está en el array
                                        }
                                    }
                                 while (arreglo[previo] < valorBuscado) {
                                    previo++;
                                    if (previo == Math.min(salto, longitud)) {
                                        return -1;
                                    }
                                }
                                    if (arreglo[previo] === valorBuscado) {
                                        return previo;
                                    }
                                    return -1; 
                                  }
